# Before run this api

### Copy-paste the .env file

```bash
$ cp .env.example .env
```

Edit the with your config 

### migration and config db

``` bash
$ yarn sequelize db:migrate
```

### Install packages

```bash
$ yarn
```

### Run

```bash
$ yarn dev
```

Server run on port 8010

### Build and run production

```bash
$ yarn build && yarn start
```

### Run test

```bash
$ yarn test
```
